# Закрывающий тег

Это личный веб-сайт портфолио, рассказывающий историю обучения веб-разработке. Сайт оформлен в стиле HTML-тегов и разделен на разделы, соответствующие различным этапам обучения.

## Структура проекта

- `index.html` - Главная страница сайта
- `styles/` - Стили CSS
  - `globals.css` - Глобальные стили
  - `style.css` - Основные стили
  - `variables.css` - CSS-переменные
  - `animations.css` - Анимации
- `scripts/` - JavaScript файлы
  - `like.js` - Функционал лайков
- `fonts/` - Шрифты и их подключение
- `images/` - Изображения
- `svg/` - SVG-графика

## Как запустить

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере

## Технологии

- HTML5
- CSS3 (с анимациями)
- JavaScript (ES6+)
